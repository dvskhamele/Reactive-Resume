## 4. Resume Builder (https://zenith-sma.netlify.app/builder)

This is the main resume editing interface with a three-column layout:

### Header
- **Logo**: Returns to dashboard
- **Save button**: Manual save of resume
- **Resume title**: Editable title field
- **Export options**: Download as PDF, JSON, or shareable link
- **Template selector**: Switch to different resume templates
- **Theme selector**: Change color scheme
- **Typography options**: Adjust fonts and text sizes
- **Layout options**: Modify section layout
- **Statistics**: Shows resume usage statistics

### Left Sidebar (Edit Sections)
- **BASICS section**: Personal information (name, headline, location, etc.)
- **WORK section**: Employment history entries
- **VOLUNTEER section**: Volunteer experience
- **EDUCATION section**: Educational background
- **AWARDS section**: Awards and honors
- **CERTIFICATES section**: Certifications
- **PUBLICATIONS section**: Publications
- **SKILLS section**: Skills and competencies
- **LANGUAGES section**: Languages spoken
- **INTERESTS section**: Personal interests
- **REFERENCES section**: Professional references
- **CUSTOM sections**: User-defined sections
- **Add Custom Section**: Create your own sections

### Main Preview Area
- **Live preview**: Real-time rendering of your resume
- **Interactive elements**: Sections can be dragged to rearrange
- **Picture section**: Upload and customize profile image
- **Responsive preview**: See how resume looks on different devices

### Right Sidebar (Configuration)

#### Information Tab
- **Summary section**: Edit the summary/introduction paragraph
- **Picture settings**: Configure image size, shape, effects

#### Export Tab
- **Export options**: PDF, JSON, PNG formats
- **Print settings**: Page size, orientation, margins
- **Filename customization**: Set export filename

#### Layout Tab
- **Drag & drop layout**: Visually arrange resume sections
- **Multiple columns**: Arrange sections in 2-3 column layouts
- **Column management**: Add/remove/reorder sections

#### Template Tab
- **Template gallery**: Browse all available templates
- **Live preview**: See how your content looks on different templates

#### Theme Tab
- **Color picker**: Customize primary, secondary, and accent colors
- **Background color**: Change background color
- **Text colors**: Adjust text color scheme

#### Typography Tab
- **Font selector**: Choose from available fonts
- **Font sizes**: Adjust different text element sizes
- **Font weights**: Modify text boldness
- **Line heights**: Adjust spacing between lines

## 5. Section Editing Details

### BASICS Section
- **Name**: Your full name
- **Label**: Professional headline
- **Image**: Profile picture upload
- **Email**: Contact email
- **Phone**: Phone number
- **URL**: Website/portfolio link
- **Location**: Address information
- **Summary**: Professional summary/introduction

### WORK Experience
- **Company**: Employer name
- **Position**: Job title
- **Location**: Work location
- **URL**: Company website
- **Start/End dates**: Employment period
- **Summary**: Job responsibilities and achievements
- **Highlights**: Bullet points for key accomplishments

### EDUCATION
- **Institution**: School/university name
- **Study Type**: Degree type (Bachelor's, Master's, etc.)
- **Area**: Field of study
- **Score**: GPA or grade
- **Start/End dates**: Education period
- **Courses**: Relevant coursework

### SKILLS
- **Name**: Skill category
- **Level**: Proficiency level (Beginner, Intermediate, etc.)
- **Icon**: Visual representation of the skill
- **Keywords**: Specific skills within the category

### LANGUAGES
- **Language**: Language name
- **Fluency level**: Proficiency description

### PICTURE Section Settings
- **URL**: Image location
- **Size**: Dimension in pixels
- **Aspect Ratio**: Maintain proportions
- **Border Radius**: Round image corners
- **Effects**: Border and grayscale options
